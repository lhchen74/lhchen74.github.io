# 注意事项

在 ON 条件里应该只写表连接条件，数据筛选放在 WHERE 里。

左外连接不筛选右边数据。

ID ID
1 1
2 2
3

左外连接使用 WHERE 筛选右边数据。

左外连接使用 ON/+ 筛选右边数据。符合 t_b.id = 2 的数据只有一条，但是左外连接需要显示左边所有数据，所以 t_a.id = 1 这条数据对应的右边数据为空。在 ON 条件里应该只写连接条件，数据筛选放在 WHERE 里。
