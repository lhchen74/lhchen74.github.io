# 问题

table1

id name description
1 a aaa
2 b bbb
3 c ccc

table2

id name description
1 x xxx
2 y yyy

通过 id 更新 table2 的 name 和 description 到 table1

result

id name description
1 x xxx
2 y yyy
3 c ccc

test data

# 方式 1

这种情况下需要添加 WHERE EXISTS (SELECT 1 FROM table2 t2 WHERE t1.id = t2.id) 只更新 table2 中存在的数据，如果不添加会将 table1 id = 3 的 name, description 更新为 null
