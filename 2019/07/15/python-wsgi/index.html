# WSGI

WSGI：Web Server Gateway Interface。
WSGI 接口定义非常简单，它只要求 Web 开发者实现一个函数，就可以响应 HTTP 请求。

上面的 application () 函数就是符合 WSGI 标准的一个 HTTP 处理函数，它接收两个参数：

  • environ：一个包含所有 HTTP 请求信息的 dict 对象；
  • start_response：一个发送 HTTP 响应的函数。
