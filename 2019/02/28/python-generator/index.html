# 栈帧

栈帧就是一个函数执行的环境：函数参数、函数的局部变量、函数执行完后返回到哪里等等。python 中一且皆对象，运行函数前首先会创建栈帧 (stack frame) 对象，然后在这个上下文环境中运行字节码对象，所有的栈帧都是分配在堆内存上，不去释放就会一直存在，这就决定了栈帧可以独立于调用者存在。
