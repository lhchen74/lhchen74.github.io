# 安装 typescript

# 初始化 typescript 环境

tsc --init 初始化 tsconfig.json

"rootDir": "./src" 设定 typescript 源文件目录为 src， "outDir": "./out" 设定编译后的 js 文件目录为 out.

# 初始化 node 环境

npm init 初始化 package.json, 安装 lodash, request , 同时安装相应的 typescript 声明文件 @types/lodash, @types/request 作为 devDependencies。

编写启动脚本 "start": "tsc && node out/index.js"
