# canvas

HTML5 canvas 元素用于图形的绘制，通过脚本 (通常是 JavaScript) 来完成，canvas 标签只是图形容器，您必须使用脚本来绘制图形。你可以通过多种方法使用 canvas 绘制路径，盒、圆、字符以及添加图像。

# canvas flow

  1. 定义 canvas 元素；
  2. 获取 canvas 元素；
  3. 请求图形的绘图上下文可以获取 2d 的和 3d 的绘图环境；
  4. 在绘图上下文上调用相应的绘制函数，以绘制图形；
