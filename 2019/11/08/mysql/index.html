# 常用参数

参数 含义
-u, --user= 用户名
-p, --password= 密码
-P, --port= 端口号
-h, --host= 服务器名称
-D, --database= 打开指定数据库
–delimiter= 指定分隔符
–prompt= 设置提示符
-V,–version 输出版本信息并退出（必须是大写 V）
